i4 SCADA-最新的SCADA軟體

100％基於HTML5 Web SCADA

i4SCADA Server IPC

i4SCADA Server IPC 平台搭載 i4SCADA 中央監控軟體平台，基於 Web 與遠端監控架構，進行生產過程控制監控與數據採集分析，i4SCADA 具備了 Web 網頁自適應畫面縮放自動排版(RWD)功能，適用於任何行動移動裝置進行生產過程監控，SCADA 系統搭載 MS-SQL express 資料庫進行數據採集，與警報紀錄歸檔數據庫應用，並具備多國語言用戶應用功能。




性能

CPU： Intel 12代 Core i5 CPU 6核心 2.9 GHz 基頻
作業系統：Windows 10 IoT 64位元
記憶體： 16G RAM
儲存空間： 1T SDD
顯示輸出 : HDMI輸出 + DisplayPort
串列通訊接口： 2 x RS232/422/485 可切換
乙太網路接口：2 x 2.5GbE + 1 x 1GbE RJ-45 LAN
工作溫度： 0°C~50°C    儲存溫度:-20°C~80°C
電源輸入： 18VDC-36VDC

* 預置安裝i4SCADA軟體



i4SCADA-SCADA軟體 特色

使用i4SCADA 監控軟體，可快速實現基於HTML5 Web的過程監控可視化效果。 直觀的用戶界面，您只需要為該工具進行短暫的培訓即可達成Web中央監控任務應用。

i4SCADA Smart Editor 智能編輯器–工程與設計

使用 i4SCADA Smart Editor，您可以通過拖放在幾分鐘內創建完全基於 Web 的過程可視化效 果，而無需任何編程知識。您只需要為該工具進行短暫的培訓即可。 符號庫包含 1000 多個符號。如果需要，可以使用自己的符號庫擴展智能編輯器。通過發佈 功能，可以輕鬆地將項目轉移給其他用戶。







i4SCADA 符號庫-輕鬆設計過程可視化

• i4SCADA符號庫當前包含1000多種不同的HTML5組件和符號。 其中包括用於過程工程和建築管理的靜態符號，帶有顏色漸變的背景以及各種通用符號。 您可以根據自己的特定要求隨時擴展現有的符號庫。

• 借助開關和按鈕，用戶可以打開和關閉聚合以及設置值。數值和字母數字值可以自由定義的顯示格式輸入和輸出。 i4SCADA檢查數值和物理單位的合理性。數值可以顯示刻度顯示，您可以在其中自由選擇比例和顏色。文本顯示的字體和样式以及背景和前景色 都是可以自由選擇的。

• 系統軟體支援開發工具與 i4SCADA Runtime Ultimate 運行工具模組，點數可支援 100Tag 數據採集與擴 展 25萬 Tag 數據採集外部控制器應用，Web Client 具多人連線架構，無連線數量限制。

另外可依 照經濟成本需求，採購 i4SCADA Runtime Standard 版本 與可選配 Client連線數




i4SCADA Studio-專業地管理項目數據庫

使用i4SCADA Studio，您可以創建，編輯和管理項目數據庫。 這包括與底層PLC控件的通訊配置，所有過程變量的創建和編輯，i4SCADA Historian和i4SCA DA報警管理器的參數設置。此外，您還可以使用i4SCADA Studio設置用戶管理，各種語言，腳本庫和附加組件。

i4SCADA是開放式SCADA系統，具有用於OPC DA和OPC UA的集成標準接口。此外， 還包括所有知名PLC控制的接口，具備SIMATIC、Modbus、OPC DA/UA、BECKHOFF ADS控制器通訊協議與， Server SDK功能。

您可以在操作過程中將過程可視化切換為所有通用語言。使用i4SCADA Studio， 您可以用項目特定的文本補充標准文本並添加新的語言，例如支援VChannels可撰寫VB Script 語言進行 資料庫連線與資料庫查詢語法。





i4SCADA群集可提供可靠的保護，防止系統故障

i4SCADA Cluster可以隨時檢查服務器應用程序，通信通道和整個系統的組件是否正常運行。發生故障時， i4SCADA Cluster會立即切換到並行運行的服務器，以確保應用程序的高可用性。您可以輕鬆地創建群集 環境並以對話框為嚮導，從而可以通過使用現有資源來避免重複使用硬件。i4SCADA群集的其他優點是： 通過熱備用和在故障情況下立即自動切換到備份服務器，以及並行數據存儲以及主服務器和備份服務器 之間的自動同步，可以不間斷地生產產品。此外，您可以通過Web前端方便地管理冗餘系統。
