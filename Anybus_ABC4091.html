Anybus ABC4091 通用乙太網路協議 - PROFIBUS DP device


Anybus 通訊協議轉換模組
通用乙太網路協議 - PROFIBUS DP device

型號:ABC4091

透過 EtherCAT、EtherNet/IP、Modbus TCP 或 PROFINET 網路與 PROFIBUS 網路在 PLC 控制系統之間無縫傳輸資料。只需連接、配置即可完成。

Anybus 通訊協議轉換模組確保不同工業乙太網路和現場匯流排網路之間可靠、安全和高速的資料傳輸。獨立 Gateway 非常易於安裝和部署，可在 PLC 之間實現透明的資料交換，使您能夠將傳統設備整合到現代高效能網路中，無需對硬體進行任何更改，只需對軟體進行最小程度的更改。

Anybus 通訊協議轉換模組採用屢獲殊榮且經過驗證的 Anybus NP40 工業網路處理器構建，可提供網路一致性、高效能和可靠性。當不同網路上的 PLC 之間連接時，可實現極快的資料循環，每個方向的資料傳輸高達 1500 字節，滿足當前大多數控制應用需求並支援未來需求。

直覺的配置、易於理解的文件以及智慧的硬體和外殼設計確保了快速安裝。

資料交換量：
Modbus TCP Server 輸入/輸出：最大各 1500 Bytes
PROFINET IO Slave 輸入/輸出：最大各 1024 Bytes
Ethernet/IP Slave 輸入/輸出：最大各 1448 Bytes
EtherCAT Slave 輸入/輸出：最大各 1486 Bytes
PROFIBUS Slave 輸入/輸出：最大各 244 Bytes

特色與優勢

優異的性能

  • 即時資料傳輸 -在兩個 PLC 之間傳輸資料所需的時間包括第一個網路上的週期時間加上第二個網路上的週期時間。Anybus 通訊協議轉換模組中的內部資料傳輸可以忽略不計因為它符合網路週期時間（抖動）的自然變化。

  • 硬體加速位元組序轉換（位元組交換）- Anybus 通訊協議轉換模組可透過硬體加速位元組序轉換來變更資料表示（位元組序），以確保資料在每個 PLC 中正確呈現。你甚至可以以不同的方式轉換資料區域的不同部分以處理不同的資料類型。它對效能沒有影響，減輕資料轉換任務的PLC負擔，簡化了PLC程式設計。

易於使用

  • 專用乙太網路配置連接埠- 無需特殊電纜 。
  • 直覺的基於Web 的拖放設定介面- 無需安裝任何其他軟體。
  • 正面連接埠使連接電纜變得容易，纖薄的外形尺寸節省了DIN導軌上的空間。
  • 透過強大的診斷功能（包括即時資料監視器、狀態螢幕和支援包)對問題進行故障排除。

最新安全功能

  • 安全啟動功能可偵測韌體竄改並抵禦惡意軟體的攻擊和感染。
  • 安全開關，可鎖定您的設定並防止任何未經授權的存取。
  • 生產中使用的連接埠已停用，以防止透過連接埠載入惡意軟體。

專為工業環境設計

  • 堅固緊湊的外殼。
  • 工業組件根據CE和UL進行測試和認證。
  • 溫度範圍寬，-25°C 至70°C。
  • DIN導軌安裝，可安裝在連接設備附近，減少接線量。
